# TW New Developer Audit

> 来源: [Shopee Open Platform](https://open.shopee.com/developer-guide/732)
> 分类: Getting Started


為保障蝦皮購物平台用戶的個人資料安全及確保開發者遵循蝦皮開放平台的資料保護政策，自即日起，針對申請台灣市場新開發者（New Developers），將實施更嚴謹的開資格審查機制。

#### **1. 適用對象**

所有欲申請台灣市場之新開發者帳號申請者。

#### **2. 審核資格**

針對台灣市場，除了一般申請資格外，您亦須符合以下資格要求：

1. 您的系統介面必須支援繁體中文選項。
1. 您必須先與其他台灣電商平台進行串接
1. 您須使用符合資安認證 (e.g., ISO 27001) 的國際雲端服務廠商，並確保主機資料存儲地位於台灣，以符合資訊安全之合規要求。
#### **3. 三階段審核流程**
1.**資格審查：**填寫開發者申請表單，並符合上述審核資格。
1.**資料安全文件審查：**提交《資料安全調查問卷》及《資料安全聲明書》以供審查。
1.**視訊訪談抽驗：**通過資料安全文件審查之申請者，系統將隨機抽驗進行視訊訪談。受抽驗之申請者須配合完成訪談程序，其結果將作為最終審核通過與否之依據。
#### **4. 注意事項**
- 由於審查流程包含多個階段，申請者於過程中可能收到階段性拒絕通知並需提交多次申請，敬請留意審查通知電子郵件。
- 審核時效可能因您的資料提交作業而延長，建議預留充足的申請時間。
- 若您因《資安調查問卷》評估結果未通過審核，同一統一編號企業最多僅可提出 三次（含首次） 申請。若三次申請皆未通過平台審核標準，本平台將不再受理同一統一編號企業之申請。
- 若您因「視訊訪談」評估結果未通過審核，同一統一編號企業最多僅可提出 兩次（含首次） 申請。若兩次申請皆未通過平台審核標準，本平台將不再受理同一統一編號企業之申請。
- 視訊訪談為必要審核程序，申請人如未出席，將視同未通過審核。
