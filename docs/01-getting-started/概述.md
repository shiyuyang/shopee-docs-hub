# 概述

> 来源: [Shopee Open Platform](https://open.shopee.com/developer-guide/4)
> 分类: Getting Started

## **Shopee Open Platform是什么？**


Shopee Open Platform是基于Shopee各类电子商务业务的开放平台，提供外部合作伙伴参与服务Shopee卖家的工具和服务。

- 强大的接口能力：Shopee Open Platform提供的丰富OpenAPI，涵盖订单、商品、营销等全方位的能力，开发者基于Shopee Open Platform开放的标准化OpenAPI接口与消息推送服务，可以快速进行定制开发，为商家搭建标准化场景应用，例如订单发货、商品管理、营销推广等。
- 全面的开发者服务：Shopee Open Platform为开发者提供从开发者入驻、开发测试到上线运维的全生命周期服务，包括但不限于应用管理、沙箱测试环境与账号、接口测试工具以及平台公告、文档中心、工单支持、站内搜索等，力求为开发者提供优雅高效的开发体验。
## **为什么需要Shopee Open API？**


Shopee Open API 让卖家可以快速轻松地访问店铺数据，管理订单、商品、营销等信息，以便在 Shopee 上进行销售。Shopee Open API 适用于所有市场，即使您服务多个市场，或者您是多个市场的卖家，您也只需要集成一套 Open API。

## **Shopee Open Platform业务接入模式**


Shopee Open Platform主要提供两种业务接入模式，包括软件服务商身份接入模式和商家自研身份接入模式，无论你是第三方开发者 (ISV) 还是商家，都能基于Shopee Open Platform找到适合自己的业务方向。

- 软件服务商身份接入模式： Shopee Open Platform提供的丰富OpenAPI，涵盖Shopee电商各个核心业务流程，如果您是有丰富的企业信息系统开发经验的传统软件厂商，您可以基于Shopee Open Platform的开放能力，为商家提供各类应用，解决店铺运营管理和营销推广等方面的问题。
- 商家自研接入模式： 针对有软件开发能力的大型品牌或商家，您可以根据自己的定制业务场景，入驻Shopee Open Platform，创建商家自用型应用，集成到自己的店铺管理系统，进行统一业务运营管理，通过对数据进行管理，发掘更多的潜在价值。
