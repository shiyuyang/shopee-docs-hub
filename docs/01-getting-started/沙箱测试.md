# 沙箱测试 V2.0

> 来源: [Shopee Open Platform](https://open.shopee.com/developer-guide/644)
> 分类: Getting Started

## 整体介绍：

（**注意：**登录 Open Console 后查看 Test Account 功能可判断是否为新沙箱，新沙箱则显示为 Test Account-Sandbox v2）


沙箱环境是 Open Platform 提供给开发者的测试环境，提供了多种类型的测试账号与数据，开发者可以在沙箱环境完成大部分 API 场景的测试工作，沙箱提供了与线上基本一致的功能，但只覆盖核心场景，例如：商品管理、订单管理、物流发货等。


**Sandbox V2支持范围：**

|**Portal**|**Features**|**Remark**|
| --- | --- | --- |
|**Console**| Shop | Create test shop account | |
| Order | Create test order | |
| Push | Push test data | |
|**Seller Center**| Product | Create and manage the global SKU | |
| Publish and manage the shop SKU | |
| Order | Order | |
| Ship Order | 暂不支持打印面单，请使用 Open API 打印 |
|**Open API**| Product | All APIs | 通过 Console 中 API Test Tools 进行调用，或者使用 https://openplatform.sandbox.test-stable.shopee.sg/域名自行调用（CN 请使用 https://openplatform.sandbox.test-stable.shopee.cn/） |
| Global Product | All APIs |
| Media Space | All APIs |
| Order | All APIs |
| Logistics | All APIs |
| First Mile | All APIs |
| Shop | All APIs |
| Merchant | All APIs |
|**Push**| 支持接收部分 push 测试数据，具体见 3.3 |


（如果有更多需要沙箱支持的功能，请提工单反馈您的具体使用场景和诉求）

3.2.3.3 MTSKU 和 MPSKU


全球商品(MTSKU)：MTSKU 相当于父产品，虚拟商品，可以发布到多个区域，生成多个产品 MPSKU。 


店铺商品(MPSKU): item=商店商品。 买家可见的真实物品。


*Note：MPSKUs 会直接继承 MTSKUs 的基本信息，多个地区的 MPSKUs 和 MTSKUs 库存信息是一样的，卖家不再需要管理多个地区的同一产品的一些基本信息。卖家只需要修改 MTSKU 的基本信息，系统可以自动同步修改所有已发布的 MPSKU。*


#### **3.2.4 全球商品订单发货**


a. 创建订单步骤与 Local Shop 相同，可以参考**3.1.3 创建测试订单 和****3.1.4 查看已创建订单。**


![图片](https://open.shopee.com/opservice/api/v1/image/download/?image_id=yZ4R3Vz6EA5RaqDPiIiqGPgNwJ%2Bc7olNpJcq7kg3hUDADueXGzTt2JD5wpHN%2FRhX3JDVbL7PtZXxaWgjSy7mBw%3D%3D&image_type=png)


b. 创建订单步骤完成后，在 Test Account-Sandbox v2> China Merchant and CB Shops 下找到您的测试账号，并通过按钮“Login Seller Center”。登陆成功后进入“My Order” 页面，并选择你创建的订单对应的站点和 Shop。


![图片](https://open.shopee.com/opservice/api/v1/image/download/?image_id=zdbevtpb3mhbV%2FUgUbBPfvNOB8AAkqpN6XvJlEsKq2gg9z7raWPS5VyRXuZh9EIQxNvknsDaHAKJ0f6GtJet8w%3D%3D&image_type=png)


c. 切换到对应的 shop 之后，您可以看到对应的订单，订单状态流转到“To Ship”后可以操作发货（在发货前请确定全球商品首公里发货仓库设置完成）。 


![图片](https://open.shopee.com/opservice/api/v1/image/download/?image_id=s1gbYcKtBEAg8W5SirGsP5kG8plU%2FtXls8qrmIZdnx2c8BnyGtTUS8Nx9EYKR9UATlSOWXdIs0IQDrk7QjZ%2BoA%3D%3D&image_type=png)


d. 选择你的订单 点击 “Arrange Shipment”。 


![图片](https://open.shopee.com/opservice/api/v1/image/download/?image_id=XuZhCqlQ5P9IoAzT7v9DKLezxT84m01oufHjRa7rx8I4dPEtJP0%2FsrX%2Fkx%2B42fbc5Fo8cbNrSGxxz%2FmyGkD5Yg%3D%3D&image_type=png)


e. 选择你的订单 点击 “Arrange Shipment”，选择 Drop-off 方式，点击"Confirm"发货。（CB 渠道目前只支持 Drop-off 方式）


![图片](https://open.shopee.com/opservice/api/v1/image/download/?image_id=crh%2BzbTN1xDjWCWUymx6xQS89FYoaqXxWf3tqT%2Fo%2B1glCdx7jkjh0tb1QYLIsRrx%2BWvjE2O561I9njYcjK2pXA%3D%3D&image_type=png)
