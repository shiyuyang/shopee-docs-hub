# Manual Mode 迁移至 GMV Max 常见问题

> **来源：** https://ads.shopee.com.my/learn/faq/505/2030
> **分类：** GMV Max

**1. 为什么要从 Manual Mode 迁移到 GMV Max？**

GMV Max 作为升级版广告解决方案，提供更广泛的流量覆盖和系统智能优化能力，能帮助卖家以更少的运营投入获取更好的广告效果。Manual Mode 逐步退出是为了让所有卖家都能享受到升级后的广告能力。

**2. 迁移时间线是怎样的？**

迁移将分阶段进行。具体时间节点请关注 Shopee 平台公告和 Seller Center 通知。在迁移截止日期前，您仍可使用 Manual Mode 创建新广告，但建议尽早开始使用 GMV Max 以熟悉新功能。

**3. 迁移过程中我的广告会中断吗？**

原有 Manual Mode 广告活动在到期前不会自动停止。但请注意迁移截止日期后可能无法创建新的 Manual Mode 广告。建议提前在 GMV Max 中创建替代广告活动，确保推广不中断。

**4. Manual Mode 和 GMV Max 的预算可以互通吗？**

广告金余额是共用的，可用于任何广告类型。但单个广告活动的预算设置是独立的。

**5. GMV Max 支持我之前在 Manual Mode 中使用的所有功能吗？**

GMV Max 不支持手动关键词选择和手动出价设置，但提供了以下增强功能：更广泛的流量覆盖；系统智能出价优化；ROAS Target 设置；Ad Group 管理；Rapid Boost 快速起量等。

**6. 如何为迁移做准备？**

- 提前熟悉 GMV Max 的创建流程和设置选项
- 记录现有 Manual Mode 广告的关键指标（ROAS、GMV、订单量等），作为 GMV Max 新广告的参考基线
- 在新品或非核心商品上先行测试 GMV Max，积累经验后再迁移主力商品
- 关注 Shopee 官方教程和指南
