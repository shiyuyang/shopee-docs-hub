# Shop GMV Max（自动选择商品）

> **来源：** https://ads.shopee.com.my/learn/faq/505/2078
> **分类：** GMV Max

**什么是 Shop GMV Max？**

Shop GMV Max（原"Automatically Select Product"）是覆盖全店的自动化广告解决方案，系统将自动选择店铺中尚未投放广告的 SKU 进行推广，通过一个广告组、一个预算、一个 ROAS 目标简化全店广告管理。

**Shop GMV Max 的运作方式：**
1. 系统自动扫描店铺所有未在投放广告的 SKU
2. 在一个广告组中集中管理多个商品
3. 共享预算和 ROAS 目标，由系统智能优化各商品间的预算分配
4. 持续自动选择新增的未推广商品

**Shop GMV Max 与 Auto Escrow Top-Up 配合使用：**
当与 Auto Top-Up (Escrow) 功能同时启用时，Shop GMV Max 广告组将享有 ROAS Protection 保障。系统将在符合条件的卖家账户中自动创建 Shop GMV Max 广告组（仅适用于尚无活跃广告组的卖家）。

**如何管理 Shop GMV Max：**
- 广告上线后，可随时修改目标 ROAS 或暂停广告活动。
- 如需修改预算，需先暂停当前广告，再通过"Create Ads"方式新建 Shop GMV Max 广告组。
- 可在 Shopee Ads 页面的"Diagnosis"板块查看系统个性化优化建议。
