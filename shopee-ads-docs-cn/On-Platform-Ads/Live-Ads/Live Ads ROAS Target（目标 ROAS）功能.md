# Live Ads ROAS Target（目标 ROAS）功能

> **来源：** https://ads.shopee.com.my/learn/faq/360/1780
> **分类：** Live Ads（直播广告）

ROAS Target 是 Live Ads 中「Maximise GMV」出价策略下的一项功能。

Shopee 将针对不同使用场景推荐 3 个 ROAS Target 值：

- **较低的推荐 ROAS Target：** 选择此项可让系统获得最具竞争力的投放效果，旨在为您的直播广告触达尽可能多的潜在买家。
- **最优推荐 ROAS Target：** 选择此项可优化广告表现，在广告花费使用率和触达更可能购买您商品的潜在买家之间取得平衡。
- **较高的推荐 ROAS Target：** 适合比较保守的广告主，设置此值有助于更好地控制广告花费。但这也可能导致广告曝光量减少、每日预算无法充分利用，甚至错失潜在买家。
- **自定义 ROAS Target：** 适合希望自行设定 ROAS 目标的卖家。

**为什么要使用 ROAS Target？**

- 通过 ROAS Target，您可以更好地掌控预算优化方向。
- 随着系统积累更多历史表现数据，ROAS Target 设置将持续优化广告表现，在较长周期内更接近您设定的目标。
- 建议在一段时间内保持相同的目标不变，以便系统更好地朝目标方向优化。

**如何设置 ROAS Target？**

创建以「Maximise GMV」为出价策略的 Live Ad 即可为其设置 ROAS Target。

**为新 Live Ad 设置 ROAS Target**

- 在 Shopee Ads 首页，点击创建新广告并选择 Live Ads
- 完成基本设置并选择 Maximise GMV 作为出价策略后，点击 Custom 设置您自己的 ROAS Target

**为已有 Live Ad 设置 ROAS Target**

- 在 Shopee Ads 首页，找到对应的 Live Ad 并点击进入 Live Ads 详情页
- 确保您已选择 Maximise GMV 出价策略以设置自己的 ROAS Target
- 对于此前已有 Maximise GMV 出价策略历史数据的 Live Ads，建议以过去 7 天的整体 ROAS 表现为基准来设定 ROAS Target 目标

**如何监控和优化 Target ROAS 表现？**

- 将 Live Ad 安排在直播时段内，确保系统在合适的时间合理消耗预算。
- 在初始阶段，系统正在学习朝您的 ROAS Target 方向优化，可能会出现一定的表现波动，这属于正常现象。
- 强烈建议在此期间（至少 7 天）不要修改 ROAS Target 值或预算设置，让系统完成优化。
- 表现稳定后，建议每月监控一次表现并更新 ROAS Target，作为常规优化操作。
- 应监控选定时间段（如过去 7 天）内的整体 ROAS 表现。
- ROAS Target 并非保证值（例如广告质量较低或 ROAS 目标设定过高时），不同 Live Ads 广告活动之间可能存在差异（如不同时段的广告竞争环境不同）。

**注意：** 此功能目前处于 Beta 测试阶段，尚未向所有用户开放。
