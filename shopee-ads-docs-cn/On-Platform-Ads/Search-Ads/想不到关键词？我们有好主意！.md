# 想不到关键词？我们有好主意！

> **来源：** https://ads.shopee.co.th/learn/faq/248
> **分类：** Search Ads（搜索广告）

**关键词创意指南——当你想不到关键词时：**

📌 **从商品本身出发：**
- 这是什么？（品类词）
- 什么材质/颜色/尺寸？（属性词）
- 什么品牌？（品牌词）
- 用在什么场景？（场景词）

📌 **从消费者角度思考：**
- 消费者会用什么词搜索这个商品？
- 他们会如何描述需求？（如"显瘦连衣裙"而非仅"连衣裙"）

📌 **利用工具：**
- Shopee 搜索下拉框自动补全
- Shopee Ads 关键词推荐工具
- 同类热销商品的标题分析

💡 **提示：** 长尾关键词（3-5个词的组合）虽然搜索量较小，但竞争更低、转化率通常更高！
