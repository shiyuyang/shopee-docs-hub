# 更轻松地为广告添加关键词

> **来源：** https://ads.shopee.co.th/learn/faq/1445
> **分类：** Search Ads（搜索广告）

**关键词添加技巧：**

为广告选择合适的关键词是提升表现的关键。以下是一些让关键词选择更轻松的方法：

1. **使用系统推荐：** Shopee 会自动推荐与您商品相关的热门关键词，优先考虑这些推荐词。
2. **搜索下拉框参考：** 在 Shopee 搜索框中输入商品品类，观察自动补全的建议词——这些都是真实的消费者搜索词。
3. **竞品分析：** 浏览同类热销商品的标题，提取高频关键词。
4. **多角度思考：** 除了品类词外，加入属性词（颜色、材质、尺寸）、场景词（户外、居家、办公）、品牌词等。

**关键词管理最佳实践：**
- 每件商品至少使用 10 个相关关键词
- 定期检查关键词表现，移除高消费低转化的关键词
- 使用 Broad Match 获取更多流量，使用 Exact Match 获取精准流量
