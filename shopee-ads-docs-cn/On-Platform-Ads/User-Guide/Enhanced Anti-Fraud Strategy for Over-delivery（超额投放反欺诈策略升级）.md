# Enhanced Anti-Fraud Strategy for Over-delivery（超额投放反欺诈策略升级）

> **来源：** https://ads.shopee.com.my/learn/faq/78/2253
> **分类：** 用户指南

为了确保 Shopee 广告生态系统的稳定性和扣费体系的公平性，平台近期升级了算法，加强了对异常广告操作的实时监控。

频繁或不当修改广告设置，不仅会打断系统算法的"深度学习"过程，还可能触发风控机制，导致账户受限。本文将帮助您了解几种典型的不合规超额投放风险行为及其后果。

**什么是"超额投放风险操作"？**

"超额投放风险操作"是指试图通过非正常干预系统的方式，在不付费或少付费的情况下获取额外流量的行为。虽然这类行为看似能在短期内钻空子，但实际上会对店铺的长期质量评分造成不可逆的损害。

典型的风险操作包括但不限于：

- **异常开关频率：** 以偏离正常商业逻辑的极高频率反复开启和关闭广告活动（Campaign）。
- **不合理的预算波动：** 在短时间内对每日预算（Daily Budget）进行大幅度、反复修改。
- **频繁更换推广商品：** 快速且频繁地修改或更换推广商品，导致投放路径极不稳定。
- **逃避系统结算：** 试图通过特定操作方法绕过正常的广告费用结算流程，造成扣费异常。

**违规操作的代价**

Shopee 广告引擎具有极高的识别精度。一旦系统判定您的账户存在恶意逃避费用或利用系统延迟的行为，平台将根据违规严重程度采取以下措施：

- **流量限制：** 系统将降低违规账户的出价排名，即使您出高价，广告也难以获得展示。
- **账户冻结：** 对于严重违规或反复欺诈的账户，平台将暂时或永久禁用其广告功能，甚至可能关停关联店铺。
- **欠费追缴：** 平台保留通过卖家钱包（Seller Wallet）扣除未支付广告费用的权利。

**广告优化的最佳实践**

我们理解卖家对 ROAS 的追求，但健康的优化应该遵循算法逻辑：

- **给系统学习的时间：** 建议每次修改后至少观察 3-7 天。频繁变更会导致算法反复重启"学习阶段"，反而使转化效果变差。
- **循序渐进调整：** 无论是调整预算还是出价，建议每次调整幅度控制在 10% 以内，避免触发风控系统的异常波动监控。
- **保持充足余额：** 确保广告账户资金充足，避免因频繁欠费导致投放中断——这会严重损害商品在算法中的广告排名积累。
